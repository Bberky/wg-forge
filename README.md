# wg-forge

[![CI](https://github.com/Bberky/wg-forge/actions/workflows/ci.yml/badge.svg)](https://github.com/Bberky/wg-forge/actions/workflows/ci.yml)
[![codecov](https://codecov.io/gh/Bberky/wg-forge/branch/master/graph/badge.svg)](https://codecov.io/gh/Bberky/wg-forge)

wg-forge is a small CLI utility aiming to solve the complexity of managing multiple [WireGuard®](https://www.wireguard.com/) configurations. It allows you to declare you network configuration using YAML specification from which specific node configuration files are generated. You can also simply generate QR codes for mobile devices and keep track of changes in your network using the diff tool.
